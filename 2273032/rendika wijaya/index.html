HTML

HTML (HyperText Markup Language) adalah bahasa markah yang digunakan untuk membuat dan mengatur struktur konten halaman web. HTML terdiri dari serangkaian elemen atau tag yang digunakan untuk menandai dan mengelompokkan elemen-elemen pada halaman web. Fungsi utama HTML adalah memberikan struktur dan arti pada konten web.

Apa itu CSS ?

CSS

CSS atau Cascading Style Sheets, yang berarti Lembar Gaya Berjenjang dalam Bahasa Indonesia. CSS adalah bahasa yang digunakan untuk mengatur tampilan dan format halaman web yang ditulis dalam HTML. Dengan menggunakan CSS, Anda dapat mengubah warna, ukuran, jenis huruf, dan tata letak elemen-elemen di halaman web. CSS memisahkan presentasi visual dari struktur konten, sehingga memungkinkan Anda untuk mengontrol penampilan situs web secara terpusat dan efisien.
